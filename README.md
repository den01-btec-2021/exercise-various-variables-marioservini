# Exercise 1.10 - Various Variables

The exercise template comes with a program template that includes the function and its call.

```plaintext
Chicken:
3
Bacon (kg):
5.5
Tractor:
None!

And finally, a summary:
3
5.5
None!
```

```plaintext
Chicken:
9000
Bacon (kg):
0.1
Tractor:
Zero

And finally, a summary:
3
5.5
Zero
```
